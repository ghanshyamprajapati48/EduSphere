# EduSphere – AI-Powered Course Marketplace

EduSphere is a full-stack EdTech platform that connects educators and 
students through a secure, scalable course marketplace. Educators can 
create, manage, and monetize courses; students can browse, purchase, 
and access content through role-specific dashboards.

## Key Highlights
- Role-Based Access Control (Admin / Educator / Student) with protected routes
- JWT + Google OAuth 2.0 authentication with forgot/reset password flow
- Razorpay integration for secure course purchases and transaction history
- Cloudinary-based media pipeline for lecture image/video uploads
- Gemini API-powered AI voice assistant for in-app navigation
- RESTful API architecture built with Express.js and MongoDB/Mongoose
- Fully responsive UI built with React.js, Redux Toolkit, and Tailwind CSS
- Deployed to production

## Tech Stack
React.js · Redux Toolkit · Tailwind CSS · Node.js · Express.js · 
MongoDB · Mongoose · JWT · Google OAuth · Razorpay · Cloudinary · Gemini API
